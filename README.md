## How to View

Simply open `index.html` in any modern web browser.

## Customization

The website is easily customizable:
- **Colors**: Modify color scheme in `styles.css`
- **Content**: Edit text in `index.html`
- **Functionality**: Add JavaScript in `script.js`
- **Images**: Replace placeholder images with your own
